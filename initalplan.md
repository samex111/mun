# SMJ MUN Website Design & Development Master Prompt

You are a senior product designer from Apple, Harvard Business School, World Economic Forum, Stripe and Pentagram.

Your task is to design and build a premium institutional website for:

SMJ MUN

(Shri Seth Mangilalji Sahu International Model United Nations)

This is NOT a student event website.

This is NOT a startup SaaS website.

This is NOT a template website.

This is NOT a card-heavy Tailwind landing page.

This is a premium international institution.

The website should feel like:

* Harvard Business School
* World Economic Forum
* United Nations
* Apple
* Ivy League Institutions
* Global Leadership Summit

---

When a school principal lands on the website they should feel:

# TYPOGRAPHY SYSTEM

Most important design element.

Typography should carry the experience.

Do NOT rely on cards.

Do NOT rely on gradients.

Do NOT rely on UI.

Typography first.

Photography second.

UI third.

Headings:

Playfair Display

Large editorial headlines.

Body:

Inter

or

Geist

Clean institutional reading.

Rules:

Very large headings.

Examples:

72px
84px
96px

Desktop.

Never make headings feel small.

---

# DESIGN PHILOSOPHY

CRITICAL

The website must follow Harvard-style editorial design.

Avoid:

* grids everywhere
* card spam
* icon spam
* feature-section layouts
* SaaS landing page patterns
* startup UI patterns

Instead use:

* huge typography
* large photography
* asymmetrical layouts
* editorial storytelling
* generous whitespace

The page should feel like a publication.

Not an app.

---

# SPACING SYSTEM

Use extremely generous spacing.

Avoid:

60px sections

Use:

120px
140px
160px

between major sections.

Allow sections to breathe.

Never cram information.

Whitespace is a feature.

# SUCCESS TEST

The final website should feel like:

30% Whitespace

20% Photography
